# Progetti ESPHome del canale

In questo repository pubblico alcuni progetti utilizzati nei miei video.

Argomenti principali:

- dispositivi basati su ESP32
- integrazione con Home Assistant
- esempi di configurazione ESPHome
- automazioni e dispositivi IoT

## Struttura del repository

Ogni progetto è contenuto in una cartella separata.

Esempi:

- cancello-wifi
- sensore-porta
- display-oled

## Video tutorial

I tutorial completi sono sul mio canale YouTube.

## Note

I file YAML possono essere copiati direttamente in ESPHome e adattati alle proprie esigenze.
